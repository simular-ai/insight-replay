# InsightReplay Project Page

Static project page for **Stateful Reasoning via Insight Replay**.

Live URL (once deployed): https://simular-ai.github.io/InsightReplay/

## Deploy

This is a static site (no build step). It works as either:

- A separate repo `simular-ai/insightreplay.github.io` (org-level page), or
- A `gh-pages` branch on `simular-ai/InsightReplay`, or
- The `docs/` folder on `main` of `simular-ai/InsightReplay` with Pages
  set to "Deploy from a branch → main → /docs".

For local preview:

```bash
python3 -m http.server 8000
# open http://localhost:8000
```
